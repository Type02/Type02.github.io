Shevrolet Сamaro

Что есть на данном сайте:

1. Активные 3 страницы.
2. Блоки, для которых отдельно прописан стиль.
3. JavaScript, внизу страницы.
4. Framework bootsrap 3 (шапка страницы, слайды).
5. Добавлен шрифт для подзаголовков.
6. Работа всех ссылок (в том числе на первоисточники).
7. Фоновая заставка.
8. Текст, разная информация.
